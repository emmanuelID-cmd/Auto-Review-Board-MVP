# Auto-Review Board MVP

Auto-Review Board is a lightweight dashboard that helps e-commerce analysts turn customer review data into actionable product insights.

## Problem

Alex, an E-Commerce Analyst, struggles to analyze sales, customer reviews, and data metrics because he lacks an automated system to track, categorize, and export data. This manual work leaves him with less time to advise his team.

## MVP

The first version of Auto-Review Board will allow users to:

- Submit product review data.
- Automatically categorize reviews.
- Update product metrics from the submitted data.
- View the updated metrics in a dashboard.
- Export a report for analysis and team discussions.

The MVP focuses on making review analysis faster and more organized, helping analysts identify product feedback trends and make better-informed recommendations.
